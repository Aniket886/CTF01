Need a hint?

Base64 encoding uses a set of 64 characters: A-Z, a-z, 0-9, +, and /

The = sign is used for padding at the end

Try using an online Base64 decoder or command line tools like:

echo "encoded_string" | base64 -d

Useful Tools:

  • CyberChef (Google it!)
  • Base64decode.org
  • Command line: base64 command
  • Python: base64.b64decode()
